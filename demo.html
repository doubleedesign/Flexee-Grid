Option 1: Apply a class to a row and the columns inside will size accordingly, wrapping onto a new line if there's more.

Default classes are "row-of-x", then if you need it to change as the screen size changes, additional classes can be added to change at a specified breakpoint. e.g. "row-of-4 tablet-row-of-8" with 8 columns inside it will be 2 rows of 4 up until the tablet breakpoint, then it will change to one row of 8. You can adjust at as many of the set breakpoints as you need.
